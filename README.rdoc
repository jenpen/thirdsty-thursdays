== Thirdsty Thursdays

An application to connect WDI-7 users. Monthly happy hour information and other events will be posted regularly

Technologies Used:
* Ruby version: 2.2.3
* Rails version: 4.2.4
* Postgres: 9.4.4.1

To deploy:
Clone this repository and run bundle install.

Upcoming Features:
* User comments
* User likes
* User rsvp for events
* Add events to calendar
* Posts forum
* Updated Desktop and Mobile UI features
